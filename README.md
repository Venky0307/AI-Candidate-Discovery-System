# AI Candidate Discovery System

## Overview

An AI-powered candidate discovery and ranking system developed for the India Runs Data & AI Challenge.

The system combines semantic embeddings, feature engineering, and hybrid scoring to intelligently rank candidates based on a given job description.

The system ranks candidates by combining:

- Semantic Search (Sentence Transformers)
- Skills Matching
- Experience Scoring
- Education Matching
- GitHub Activity
- Profile Completeness

## Tech Stack

- Python
- Sentence Transformers
- PyTorch
- Pandas
- Scikit-learn
- Streamlit

## Features

- Semantic candidate-job matching using Sentence Transformers
- Hybrid ranking algorithm combining semantic and structured features
- Skill and experience based candidate scoring
- AI assessment score integration
- Explainable candidate ranking
- Automatic Top-100 candidate selection
- Challenge-compliant submission generation
- Streamlit web application

## Dataset

This project was developed using the **India Runs Data & AI Challenge** dataset.

Due to GitHub's file size limitations, the following large files are **not included** in this repository:

| File | Reason |
|------|--------|
| `candidates.jsonl` | Original dataset (~487 MB) |
| `models/candidate_embeddings.npy` | Generated candidate embeddings (~154 MB) |
| `output/candidates_processed.csv` | Generated during preprocessing |
| `output/ranked_candidates.csv` | Generated after ranking |

These files can be regenerated by running the preprocessing and embedding pipeline provided in this repository.

### Reproducing the Results

1. Place the challenge dataset (`candidates.jsonl`) in the project root.
2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

3. Generate candidate embeddings:

```bash
python test_embeddings.py
```

4. Rank candidates:

```bash
python test_ranker.py
```

5. Generate the submission:

```bash
python test_submission.py
```

The final submission will be generated as:

```
output/submission.csv
```

## Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Generate embeddings:

```bash
python test_embeddings.py
```

Generate rankings:

```bash
python test_ranker.py
```

Run the application:

```bash
streamlit run app.py
```
